# 核心 Meta-Prompt — 指令產生指令

> 這是 My Prompt Maker 的主要工具。將以下 Meta-Prompt 複製進任何 AI 對話框，填入場景與範例後，AI 會自動依分類架構輸出完整的提示詞集合（YAML + 自然語言雙格式）。

---

## 使用說明

**3 步驟快速開始：**

1. 複製下方「Meta-Prompt 本體」全文
2. 填入兩個欄位（用花括號 `{}` 標記的位置）：
   - `{場景描述}`：你要生成提示詞的使用場景
   - `{自然語言範例}`：一個你認為品質好的參考提示詞
3. 貼入 Claude / Gemini / ChatGPT 任一對話框，送出

**預期輸出：**
- 7 個分類（A~G）× 2 種格式（YAML + 自然語言）= 約 14 個輸出區塊
- 每個分類風格明顯不同，可直接使用或微調

**填入範例（NotebookLM 簡報背景）：**
- 場景：`NotebookLM 知識型簡報的背景底圖生成，比例 16:9，供商業受眾簡報使用`
- 範例：貼入使用者提供的霓虹青科技風格提示詞

【輸出要求】

針對上述 A–G 七個分類，各生成一個獨特的創意提示詞，每個分類的主題與風格必須明顯不同（不能都是同一種科技感）。

每個分類輸出以下格式：

```

## 進階使用技巧

### 針對不同 AI 優化輸出

如果你想要針對特定 AI 最佳化，在【使用場景】後面加上：

- **Claude 優化**：「請使用 XML 標籤結構（`<instructions>`, `<context>`, `<examples>`）輸出提示詞」
- **Gemini 優化**：「請確認指令放在提示詞末端，前段放背景資訊與示例」
- **Midjourney 優化**：「自然語言版本末尾加上 Midjourney 參數列，如 `--ar 16:9 --style raw --v 6.1 --quality 2`」

### 更換使用場景

將 `{場景描述}` 換成你的需求，例如：
- 「LinkedIn 個人品牌頁面的視覺 Banner，1584×396 像素」
- 「YouTube 頻道封面，科技教育主題，2560×1440 像素」
- 「企業年報封面，正式商業風格，A4 比例」
- 「Instagram 限時動態背景，直式 9:16，Z 世代受眾」

### 擴充分類架構

如果你的場景需要更多分類（例如加入「品牌識別」或「情緒氛圍」分類），在架構表中新增一行，Meta-Prompt 會自動為新分類生成提示詞。
